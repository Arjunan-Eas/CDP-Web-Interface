Format OUT of web portal (WEB_CDP):

DUID:8 character DUID_TOPIC:message_type_DATA:message_DUCKTYPE:ducktype_

ex: 'DUID:MAMA0001_TOPIC:status_DATA:This is an example message_DUCKTYPE:MAMA_


Format IN to web portal (CDP_WEB):

SDUID:8 character source device user ID_TOPIC:message_type_DATA:message_

ex: 'SDUID:MAMA0002_TOPIC:status_DATA:This is an example response_
